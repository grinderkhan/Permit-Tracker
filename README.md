# Construction Permit to Work (PTW) Management System

Professional Permit Management System for Construction Projects.

Features:
- Permit Creation
- Permit Register
- Dashboard
- PDF Export
- Excel Export
- QR Code
- Audit Trail
- Role Based Access
- Supabase Database
- Mobile Friendly

Technology:
- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
